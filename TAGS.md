# Tag Taxonomy

This is the single source of truth for project tags. When tagging a new project
(in its `.qmd` front matter under `categories:`), use values from this list only —
this keeps the homepage's tag filter clean and consistent.

To add a new tag: add it to the list below, then use the exact same string in
a project's `categories:` front matter.

## Project Series

Separate from skill tags above - these group projects that are related to
each other (e.g. multiple projects built on the same dataset/theme). Shown
as a light-brown chip alongside the status badge on homepage carousel items.

- One-Off (a standalone project, not part of a series)
